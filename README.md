# snake_game_python
A simple Snake Game built using Python and the turtle graphics module.
This project is beginner-friendly and helps in understanding core programming concepts like loops, functions, conditionals, keyboard events, and collision detection.

The player controls the snake using keyboard keys, eats food to grow longer, and earns points. The game ends when the snake hits the wall or its own body.
